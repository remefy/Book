0. Создать класс Book со свойствами:
    - id,
    - name,
    - year,
    - author,
    - genre

1. Создать класс BooksDBAdapter, в котором есть:
    - метод prepare() - подготовильные
    - метод get_all_books() - возвращает список всех книг
        как список объектов класса Book
    - метод save_new_book() - принимает одну книгу
        и сохраняет в БД
    - метод get_book_by_id(id) - по ID возвращает объект Book
    - метод delete_book(id) - удаляет книгу из БД по ID
    - метод update_book(book) - обновляет уже
            существующую книгу

    и этот класс должен хранить книги просто в памяти,
    ни в Файле, ни в БД.

2. Создать класс BooksApp,
    - который имеет метод start() - запускает программу
    - использует объект класса BooksDBAdapter()
        который передается ему в конструкторе
    - после запуска ждет команду
    - exit - выход
    - help - показывает список команд
    - show all - показывает список всех книг
    - add - добавляет новую книгу
    - delete - удаляет книгу
    - update - обновляет свойства книги
    - show - показывает свойства книги по ID

     Пример кода:
     adapter = BooksDBAdapter()
     app = BooksApp(adapter)
     app.start()

3. Унаследовать от класса BooksDBAdapter() и создать класс
    SqliteBooksDBAdapter()
        - этот класс имеет все методы BooksDBAdapter
        - но все делает в базе данных sqlite3
        - в конструкторе принимает название файла БД

4. Унаследовать от класса BooksDBAdapter() и создать класс
    ORMBooksDBAdapter()
        - этот класс имеет все методы BooksDBAdapter
        - но все делает в базе данных через ORM Peewee
        - в конструкторе принимает название файла БД
